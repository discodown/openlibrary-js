Usage Examples

Make a Request Using ReadAPI

To make a single request, use getSingleRequest() with a single identifier and its type:

                
                    const read = new ReadAPI();
                    const item = read.getSingleRequest("0596156715", "isbn");

To make multiple requests at once, pass a list of Request objects to getMultiRequest():

                
                    const requests = [new Request([new LibraryID("1", "id"), new LibraryID("50006784", "lccn")]), 
                          new Request([new LibraryID("OL6179000M", "olid"), new LibraryID("55011330", "lccn")])];

                    const items = read.getMultiRequest(requests);

A Request object is just a list of LibraryID objects, which are just an identifier and its type. You can add a library ID to a request either by using the request object's addLibraryID() method with a LibraryID object or its addID() method with an identifier and type: 

                
                    let request = new Request();
                    const newID = new LibraryID("321", "id"); 
                    request.addLibraryID(newID);
                    request.addID("oclc", "27082333");

Search Using SearchAPI

To use SearchAPI's search() method, create a QueryField object from the field you want to search and the query value. For example, to search for an author named "zola": 

                
                    const search = new SearchAPI(); 
                    const q = new QueryField("zola", "author");
                    const results = await search.search(q);
