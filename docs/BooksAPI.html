Class: BooksAPI

BooksAPI(format, jscmd, callback)

new BooksAPI(format, jscmd, callback)

Implements the Books API for fetching information about books using their ISBN, OCLC, LCCN, or OLID identifiers.
Parameters:
Name Type Default Description
format string javascript Format to return results in. Possible values: javascipt, json Default: javasctipt
jscmd string viewapi Information to provide for each identifier match. Possible values: viewapi, data Default: viewapi
callback string null Optional callback method to wrap results in. Default: null
